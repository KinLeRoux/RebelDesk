# DeskMemory for Windows

**Your Markdown vault on disk. Meaning in search. AI when you need it.**

DeskMemory is a local-first knowledge app: your notes stay as files on your machine, with semantic search, a prompt compiler, and account-backed AI chat. Manage your plan and usage at **[deskmemory.app](https://deskmemory.app)**.

Built by **[rebelGlitch](https://deskmemory.app)** — *ship over credentials.*

## Quick start

1. **Download** the installer from [Releases](../../releases/latest) (or [deskmemory.app/download](https://deskmemory.app/download)).
2. **Run** the setup wizard.
3. **Sign in** with your DeskMemory account (or [create one free](https://deskmemory.app/signup)).
4. **Choose your vault folder** — your Markdown library stays on your disk.

## Windows security notice (read this once)

This build is **indie software from a new publisher**. Windows or Chrome may show a warning such as *“suspicious file”* or SmartScreen *“Unknown publisher”*.

That’s normal for unsigned or newly published apps. The file in **Releases** matches what we host on **deskmemory.app**.

**Chrome:** Downloads → ⋮ on the file → **Keep anyway**  
**Windows:** **More info** → **Run anyway**

We’re working toward **Microsoft Store** distribution and code signing so this goes away over time. Prefer fewer warnings? Use the [official download page](https://deskmemory.app/download) or the Store listing when it’s live.

## What you get

- **Local-first vault** — Markdown + YAML on disk; you own the files  
- **Semantic search** — find notes by meaning, not just keywords  
- **AI chat** — through your DeskMemory account (usage & billing on the web)  
- **Private by design** — the vault is not uploaded to our servers  

**Free tier available.** Upgrade plans and message packs: [deskmemory.app/pricing](https://deskmemory.app/pricing)
